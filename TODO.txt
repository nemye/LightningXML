2. Generate the compile-time required mask

Add this near your other metadata helpers:

template <typename T, size_t... I>
constexpr auto make_required_mask_impl(std::index_sequence<I...>) {
  constexpr size_t N =
      std::tuple_size_v<decltype(XmlMetadata<T>::fields)>;

  std::bitset<N> mask;

  ((std::get<I>(XmlMetadata<T>::fields).required
        ? mask.set(I)
        : mask),
   ...);

  return mask;
}

template <typename T>
constexpr auto make_required_mask() {
  return make_required_mask_impl<T>(
      std::make_index_sequence<
          std::tuple_size_v<decltype(XmlMetadata<T>::fields)>>{});
}

6. Handle attributes too

Right now attributes are parsed before elements:

dispatch_attrs<T>(*this, object, kIdxSeq);

Modify apply_attr to report success:

template <typename T, size_t I>
static void apply_attr(Parser& p,
                       T& obj,
                       size_t& pos,
                       std::bitset<
                         std::tuple_size_v<
                           decltype(XmlMetadata<T>::fields)>>& parsed)
{
  constexpr auto& f = std::get<I>(XmlMetadata<T>::fields);

  if constexpr (f.kind == FieldKind::Attr) {
    if (p.attr(f.hash, obj.*(f.member), pos)) {
      parsed.set(I);
    }
  }
}

Then pass parsed into dispatch_attrs.

This gives you:

compile-time field count
compile-time required mask
zero string comparisons
no extra metadata tables
runtime cost = a couple of bit operations per parsed field

For small XML objects the optimizer will usually reduce the whole std::bitset to an integer mask. For larger objects it becomes a fixed-size bit array.
